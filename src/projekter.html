CAMPAIGN FOR BLAAGAARD THEATER

For this case, the task was to prepare a campaign targeted at the target group 'young adults'. The campaign should promote Blaagaard Theater and contain a supporting concept idea. In addition, a campaign video with a focus on storytelling and high value production had to be produced. The campaign had to be presented on a website that was stylistically true to BLAAGAARD theatre's visual ID.

The site had to convey the campaign's message, as well as involve the target group via some form of user interaction. In this project I was responsible for the production of campaign videos, including storytelling and high value production. Furthermore, I participated in the design process, in connection with the preparation of the website in accordance with BLAAGAARD theatre's visual ID.

REDESIGN OF NØRREBRO UNITED

For this case, the task consisted of developing and designing a web solution, with a focus on optimizing the user experience. The purpose of the website was to convey Nørrebro United's work and provide an overview of the many types of different game plans, teams, etc.

The website had to be created in Wordpress, where a representative number of team types had to be obtained via the WP REST API. In this project, my primary task was to perform tests and collect data with a view to creating a user-friendly structure and design.

REDESIGN OF ANE SCHOLLERT

As an exam in the 2nd semester, a multimedia solution had to be prepared for an optional Danish artist. The solution had to promote the customer's values and products, as well as illustrate the principles of navigation, functionality, ease of use, etc.

The website was to be created in Wordpress, where selected content was to be obtained via the WP REST API. In addition, content had to be prepared for both SoMe and the website, with a focus on high value production.

My group and I worked with ceramicist Ane Schollert based on the following problem: How can we improve the company's website with a multimedia solution, create a consistent expression on all platforms and attract more younger customers.

In this project I was responsible for creating content for both SoMe and the website, in the form of product images. In addition, I was responsible, in collaboration with other group members, for research and testing, with a view to preparing a user-friendly and consistent design in accordance with trends and trends within the industry.
